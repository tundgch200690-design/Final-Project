## Quick Scene Setup

### 1. Create MainMenu Scene
- File ? New Scene
- Save as "MainMenu"

### 2. Add Canvas & EventSystem
- Right-click in Hierarchy ? UI ? Panel
- Canvas created automatically

### 3. Add Main Menu Manager
- Create empty GameObject "MainMenuManager"
- Drag MainMenuManager script onto it
- Configure scene names

1. Load MainMenu scene
2. Click PLAY ? See GameModePanel
3. Click SINGLE PLAYER ? Goes to CharacterSelection

5. Adjust sliders ? Click APPLY
6. Back in MainMenu, click CREDITS ? See credits with scrolling
7. Click QUIT ? Game exits (or stops in editor)

## Common Issues & Fixes

| Issue | Fix |
|-------|-----|
| Buttons not clickable | Check Canvas ? Render Mode = Screen Space - Overlay |
| Panels overlap weirdly | Use same Canvas parent for all panels |
| Audio not playing | Assign AudioClips to MainMenuAudioManager |
| Scene won't load | Check scene names in MainMenuManager Inspector |
| Settings not saving | Verify PlayerPrefs.Save() is called in SettingsMenuUI |
| UI looks stretched | Set Canvas Scaler ? UI Scale Mode = Scale With Screen Size |
